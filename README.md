# Goal Tracker 

A simple app to manage your goals with Red Mode UI.  

## Features
- Add, edit, delete, and complete goals  
- Show progress of each goal with a progress bar  
- Red UI with white text 
- Responsive for desktop and mobile  
- Store data in browser (Local Storage)
